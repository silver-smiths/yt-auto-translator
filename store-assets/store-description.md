## 한국어 (ko)

### 요약 (132자 이내)
```
Google Gemini AI로 YouTube 스튜디오 자막을 26개 언어로 자동 번역·등록. API 키만 있으면 바로 시작.
```
> 현재 글자 수: 48자

### 상세 설명
```
YouTube 자동 번역기는 YouTube 스튜디오의 영상 자막을 Google Gemini AI를 이용해 자동으로 번역하고 등록해주는 Chrome 확장 프로그램입니다.

번역 작업을 언어별로 일일이 반복하던 번거로움을 없애고, 클릭 한 번으로 최대 26개 언어의 자막을 한꺼번에 처리할 수 있습니다.

【주요 기능】
• 26개 언어 동시 번역 — 영어, 일본어, 중국어, 스페인어 등 주요 언어를 한 번에 처리
• Gemini AI 번역 — Google의 최신 AI 모델(Gemini 2.5 Flash 등)로 자연스러운 번역 품질
• YouTube 스튜디오 직접 등록 — 번역 결과를 자동으로 자막 트랙에 업로드
• 번역 이력 관리 — 이전 번역 결과를 사이드바에서 바로 확인
• 간편한 설정 — Gemini API 키 입력 후 즉시 사용 가능

【사용 방법】
1. Google AI Studio(aistudio.google.com/apikey)에서 Gemini API 키를 발급받습니다.
2. 확장 프로그램 설정 페이지에서 API 키를 입력합니다.
3. YouTube 스튜디오의 영상 수정 페이지로 이동합니다.
4. 사이드바에서 번역할 언어를 선택하고 [번역 시작]을 클릭합니다.
5. 번역이 완료되면 선택한 언어의 자막이 자동으로 등록됩니다.

【참고 사항】
• Gemini API 사용에 따른 비용은 Google의 요금 정책을 따릅니다. (무료 티어 제공)
• API 키는 사용자의 브라우저에만 저장되며 외부로 전송되지 않습니다.
• 자막이 없는 영상에는 사용할 수 없습니다. 먼저 원본 자막을 등록해 주세요.
```

## 영어 (en)

### 요약 (132자 이내)
```
Auto-translate & upload YouTube Studio subtitles into 26 languages using Google Gemini AI. Just add your API key and go.
```
> 현재 글자 수: 119자

### 상세 설명
```
YouTube Auto-Translator is a Chrome extension that automatically translates and uploads subtitles in YouTube Studio using Google Gemini AI.

Stop manually repeating the same translation process for every language. With a single click, translate your subtitles into up to 26 languages at once.

【Key Features】
• 26-language batch translation — English, Japanese, Chinese, Spanish, and more, all at once
• Gemini AI quality — Powered by Google's latest models (Gemini 2.5 Flash, etc.) for natural, accurate translations
• Direct upload to YouTube Studio — Translated subtitles are automatically pushed to your subtitle tracks
• Translation history — Review past translation results anytime from the sidebar
• Easy setup — Enter your Gemini API key and start immediately

【How to Use】
1. Get a Gemini API key from Google AI Studio (aistudio.google.com/apikey).
2. Enter your API key in the extension's settings page.
3. Open a video editing page in YouTube Studio.
4. Select target languages in the sidebar and click [Start Translation].
5. Once complete, subtitles are automatically uploaded for each selected language.

【Notes】
• Gemini API usage may incur costs depending on your usage. A free tier is available.
• Your API key is stored only in your browser and is never sent to our servers.
• The video must already have source subtitles. This extension translates existing captions — it does not generate them from scratch.
```
