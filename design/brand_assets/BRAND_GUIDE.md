# Brand Guide: Seamless Intelligence (Plan B)

## 1. Core Concept
- **Concept Name:** "Seamless Intelligence" (심리스 인텔리전스)
- **Target Message:** Professional, Reliable, High-Tech, AI-driven.
- **Visual Goal:** High-contrast premium dark theme that minimizes eye strain and emphasizes "AI Progress."

## 2. Color Palette
| Role | Color Code | Usage |
|------|------------|-------|
| **Primary (AI)** | `#BB86FC` (Lavender) | Primary buttons, active icons, brand accent. |
| **Secondary (Sync)** | `#03DAC6` (Teal) | Success states, completion indicators, sync status. |
| **Surface-0** | `#121212` | Main background of popup and options page. |
| **Surface-1** | `#1E1E1E` | Section backgrounds, card containers. |
| **Surface-2** | `#2C2C2C` | Hover states, secondary buttons. |
| **On-Surface-High** | `rgba(255, 255, 255, 0.87)` | Headlines, primary text. |
| **On-Surface-Med** | `rgba(255, 255, 255, 0.60)` | Descriptions, labels, footer text. |
| **Error** | `#CF6679` (Soft Red) | Error states, stop buttons, warnings. |

## 3. Typography
- **Primary Font:** `Inter`, `Segoe UI`, `-apple-system`.
- **CJK Fallback:** `Noto Sans KR`.
- **Hierarchy:**
  - `Title`: 16px, Semi-bold (High-opacity white)
  - `Body-Large`: 14px, Medium (High-opacity white)
  - `Body-Small`: 12px, Regular (Med-opacity white)
  - `Label`: 10px, Semi-bold (All caps for logs/types)

## 4. UI Components Style
- **Buttons:** 12px corner radius. Primary uses solid lavender with dark text. Danger/Secondary uses outlined style.
- **Progress Bar:** Gradient from Lavender (`#BB86FC`) to Teal (`#03DAC6`).
- **Cards/Chips:** Subtle 1px border (`rgba(255, 255, 255, 0.12)`) and Surface-1 background.

## 5. Logo Concept
- **Symbol:** A minimalist "T" (for Translation) combined with a "Sync" or "Pulse" motif.
- **Gradient:** 45-degree linear gradient using Primary and Secondary colors.
- **Icons:** Modern outline style with consistent 2px stroke width.
